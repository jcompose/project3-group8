Welcome to the Real Estate API!

Use the API endpoints below to query real estate data. Or don't. It's your life.

  • Get ALL Property Data: /api/real_estate
  • Get a Property by ID: /api/real_estate/<id>
  • Filter Properties by City: /api/real_estate/city/<city_name>
  • Filter Properties by Price Range: /api/real_estate/price/<min_price>/<max_price>
  • Properties by Year Built: /api/real_estate/yearbuilt
  • Property Counts by Zip Code: /api/real_estate/zip
  • Average Price per City: /api/real_estate/avg_price/city
  • Feature Analysis: /api/real_estate/feature_distribution
  • Property Type Distribution: /api/real_estate/property_type_distribution
  • Map Visualization Data: /api/real_estate/map
